# Suporte commands vim to helper

this documentation is mine support in never forget (or no)

'esc' exit current mode 
'y' copy 
'v' selection 
'p' paste 
'c' crop
'x' delete char
'dd' delete line
'option+u' rollback last change
'option+r' rollback of rollback
'w' move skip word
'b' move back word
